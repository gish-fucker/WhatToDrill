Start

推荐使用流程

1. 记录今天状态

先填睡眠、饮水、精力、酸痛和疼痛，今日建议会按恢复状态调整。

2. 载入或记录训练

从今日建议开始，保存真实组数、重量、次数和 RPE。

3. 每周看复盘

复盘中心会整理风险、下周行动和可导出的 Markdown 周报。

Cloud advice

是否允许生成云端建议？

云端建议会发送最近的状态、训练、动作组数据和有限长度备注。不会发送本地记录 ID、更新时间、习惯对象或完整动作库。

AI 建议仅用于训练与恢复参考，不是医疗诊断。你可以继续使用本地建议，也可以稍后在帮助页撤回授权。
