Privacy

隐私政策

生效日期：2026 年 7 月 11 日

我们如何处理数据

本应用采用本地优先设计。日常状态、训练记录、动作库、模板、偏好、每周计划训练日及其生效历史和建议历史默认保存在你当前浏览器的本地存储中，不会因为普通记录操作自动上传到服务器。

导入、导出与删除

你可以主动导出 JSON 完整备份、CSV 摘要或 Markdown 周报。导出的文件由你选择保存位置并自行管理。导入 JSON 前，应用会校验并展示预览；确认后才覆盖当前浏览器的数据。你也可以在应用内清空全部本地数据。

云端建议

只有在部署者配置云端建议服务、你主动点击并明确同意后，生成建议所需的近期状态、训练和有限长度备注才会发送到服务器并转交给配置的 AI 服务提供方。记录 ID、更新时间、习惯对象和完整动作库不会随建议请求发送。你可以在帮助页撤回授权；未配置或未同意时，应用使用本地规则生成建议。请不要在记录或备注中填写身份证号、联系方式等不必要的敏感信息。

安全与技术数据

服务器会在内存中临时使用请求来源网络地址进行频率限制，以降低接口滥用风险；应用本身不会把该地址写入用户记录或备份。部署环境、网络服务或 AI 服务提供方可能按其各自政策处理必要的技术日志。

保存期限与用户控制

本地数据会保留到你清空站点数据、在应用内删除、卸载相关浏览器数据或设备存储被清理为止。更换设备或浏览器不会自动同步，请在迁移前导出 JSON 备份。清除数据后，本应用没有云端副本可替你恢复。

未成年人

未成年人应在监护人知情和指导下使用本应用，尤其是在记录健康状态或依据建议安排训练时。

政策更新与联系

功能或数据处理方式发生变化时，本政策会更新生效日期和相关说明。隐私问题请通过你获取本应用地址时由部署与提供者给出的联系渠道反馈。

关怀摘要与主动分享

关怀摘要在当前设备上由汇总数据生成。默认不包含风险提醒，也不会包含姓名、备注原文、动作明细、记录 ID 或完整历史。只有你主动选择内容并调用系统分享或复制后，摘要才会离开应用；接收方和所用分享服务对已发送内容的后续处理不受本应用控制，请在发送前核对预览。

训练协作简报

训练协作简报在当前设备上生成，用于与教练确认训练目标、环境、计划节奏、计划兑现和执行概览。你可在分享前选择计划、兑现和执行三类信息。简报不会包含健康状态、疼痛、睡眠、备注原文、动作明细、重量、次数、具体日期或完整历史；只有你主动分享或复制后，所选汇总信息才会离开应用。

个人进展报告

个人进展报告仅在当前设备根据最近阶段的训练次数、完成组数、状态覆盖、睡眠、疼痛和计划节奏生成。它不会保存新的个人资料，也不会自动上传或分享。导出的报告不包含备注原文、动作名称、重量、次数、具体训练日期、记录 ID 或完整历史；只有你主动导出或复制后，文本才会离开应用。

支持约定与回访

你可分别建立最多 6 位支持伙伴。每位约定只保存支持对象类型、关怀频率、支持方式、沟通边界、下一日期及最多 12 次日期与 1 至 5 的支持感评分；不要求姓名或联系方式，不记录聊天内容或对方信息。它们随本地数据和 JSON 备份保存；只有你主动分享邀请时，被选伙伴的结构化约定才会离开应用。若该伙伴选择“提醒我完成计划”且你设置了计划训练日，邀请会包含星期安排；邀请不会附带健康、训练明细或回访评分。
